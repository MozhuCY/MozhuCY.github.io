# 2019-7-29

C++ lambda (函数式,闭包

这也是之前看了的但是没有记录的东西,在这里写一下

```c++
auto func = []() ->  void
{

}
```

差不多这样,\[\]里面可以是=或者&用来表示对此区域变量的取值,还可以\[=,&a\],对其他取值,a取引用,()里面放参数,然后返回值,函数体,编译时要记得加std=c++11
